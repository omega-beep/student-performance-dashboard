# Student Performance Dashboard

> A full-stack web application built with Flask to manage and analyze student performance data.

---

## Features
- Secure user authentication with password hashing
- Add and delete student records
- Automatic grade and pass/fail calculation
- Search by name and filter by grade
- Interactive charts for performance analysis
- storage using SQLite

---

## Tech Stack

- backend: flask, python
- frontend: html, css, js
- database: sqlite
- visualization: chart.js
- authentication: werkzeug, flask sessions
---

```bash
git clone https://github.com/omega-beep/student-performance-dashboard.git
